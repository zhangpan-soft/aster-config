# Changelog

## 0.1.0 - 2026-05-12

Initial public release.

- Embeddable admin canvas for JSON, YAML, XML, and properties sources.
- Normalized key/value storage model with `env + namespace` identity.
- Local file profile for development and demos.
- JDBC integrated profile for business-system production integration.
- Client HTTP API and Spring Boot client starter.
- Netty revision notification with HTTP reload fallback.
- Cluster peer notification and JDBC publish-event compensation.
- MySQL schema and automated JDBC integration coverage.
